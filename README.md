# Bright-Coffeeshop-Sale
Analysis for CEO to understand sales at bright coffee shop for the past 6 months

# Proble Statement
The business has seen a decline in revenue over a period of 6 months and the new CEO wants to understand the contributing factors to the decline in revenue. As a Junior Data Analysist I am delegated to collect and analyse data and perform a diagnostic analysis to better understand the probable cause. 

# Aim
The aim of the dialgnostic analysis is to study what influences customer purchasing performance, product based performace, store location and sales trends to identify key metrics that contribute to the incline or decline of revenue. The analysis will be used to investigate probable cause to the decline in revenue and highlight key strategic points that the business will need to focus on to drive revenue in a positive incline. 

# Objective
In order to achieve this aim the following tools will be used: 
- Mirro: To be used for project planing and outline i.e. Project Flow Chart
- Canva: To be used to create project times lines >> Gunt Chart
- Databricks: To be used for data processing
- Excel: To be used for data analysis (Pivot Tables and graphs)
- Canva/PowerPoint: To be used to create a presentation for findings and recommendations to the CEO
